Context Adaptive Deep Neural Networks
=====================================


This project's objective is to voice recognition using CADNN.


## Requirements
* Ubuntu 18.04
* Python ~> 3.7.0


## Installation
```sh
$ git clone <this repo>
$ cd <this repo>
```
